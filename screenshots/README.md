## API Execution Screenshots

This folder contains screenshots captured during manual API testing of
CRUD operations using Postman on the Restful Booker API.

The screenshots serve as execution evidence to demonstrate that the APIs
were tested manually and responses were verified as per expected behavior.

### Included Screenshots
- POST – Create Booking (booking ID generation)
- GET – Retrieve Booking Details
- POST – Generate Authentication Token
- PUT – Update Booking Details
- DELETE – Delete Booking
- Negative Scenario – Unauthorized Access (403 Forbidden)
